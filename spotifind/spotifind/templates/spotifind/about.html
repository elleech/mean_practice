About the site

When we listen to a song for the first time, we usually don't know the title or the artist. All we can remember is one or two catchy lyrics. The current Spotify search only supports searches on title, artist, album, and playlist. Therefore, we plan to implement lyrics search engine for Spotify. So in the future we can look up the tune that we hear in a shop or on a street, and add it to our Spotify playlist!

This website is for the final project of INFSCI 2140 Information Retrieval. We first use Genius API to get our target corpus in real-time. (Becuase everything is the free source of the Internet, it takes time to process.) In the meantime, we use Spotify API to search for the corresponding track, and store the song data from Genius and the url link from Spotify in the database. Finally, we perform index ranking based on the user query and show the top 10 songs for our user.

We use Django for the frontend, python for the backend, and MongoDB Atlas as our database. The index library that we use is python whoosh. Our website logo (favicon) is a resource from the Internet. It is designed by @elamir.ammar.
